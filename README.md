# AbuseIPDB Spam Reporter

WordPress-плагин для автоматической отправки IP-адресов спамеров в базу AbuseIPDB. 

## Принцип работы
Плагин перехватывает комментарии, получающие статус "Спам" (вручную администратором или автоматически через фильтры вроде Akismet), и асинхронно отправляет отчет через API v2 сервиса AbuseIPDB.

## Установка и настройка
1. Скачайте файл `ab_report.php` и загрузите его в директорию `/wp-content/plugins/`.
2. Активируйте плагин в панели управления WordPress.
3. Перейдите в **Настройки -> Общие**.
4. В поле **AbuseIPDB API Key** вставьте ваш ключ (получить можно на [abuseipdb.com](https://www.abuseipdb.com/account/api)).

## Технические детали
* Использует `wp_remote_post` с параметром `blocking => false` для предотвращения задержек при обработке комментариев.
* Пишет метаданные (`_abuseipdb_reported`) для исключения дублирующих отправок.
* Категории репортов по умолчанию: `10` (Web Spam), `21` (Web App Attack).
